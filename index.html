brat by presidentbeef

Brat is a little language for people who don't like to be told what to do.

Brat is a little toy language that doesn't care what you think of it. It won't admit it, but it is not even out of infancy. Not even a toddler yet. But it can already run methods and create objects and has arrays and hashes and numbers and that sort of stuff, so it thinks quite highly of itself. Brat uses a PEG parser written using TreeTop, a Ruby parser generator. The Brat code is then converted into Neko, compiled to Neko bytecode, and then run on the Neko VM. Brat is flexible enough that you can get by with a very small core and write any functionality that most languages use keywords for. For example, you can write and use a while loop like this: my.while = { block | true? block, { while ->block } } n = 1 while { p n n = n + 1 n < 10 }
